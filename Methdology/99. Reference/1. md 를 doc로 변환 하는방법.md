# ✅ 3️⃣ 가장 기본 변환 (초기 테스트용)

같은 폴더에:

test.md

있다고 가정하면:

pandoc test.md -o test.docx

끝.

---

# ✅ 4️⃣ 지금 당신이 쓰는 실무 명령어 (핵심 ⭐)

우리가 만든 template 적용:

pandoc input.md -o output.docx \  
--reference-doc=reference_template_for_md_to_doc.docx \  
--toc \  
--number-sections

---

## 옵션 설명 (진짜 필요한 것만)

### ✔️ `-o output.docx`

출력 파일 이름

---

### ✔️ `--reference-doc`

--reference-doc=reference_template_for_md_to_doc.docx

👉 Word 디자인 적용

- Cover 스타일
    
- Heading 폰트
    
- Page Number
    

---

### ✔️ `--toc`

자동 목차 생성

---

### ✔️ `--number-sections`

Heading 자동 번호

예:

1 Problem  
1.1 Background

---
