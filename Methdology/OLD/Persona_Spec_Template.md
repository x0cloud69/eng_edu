# Example: Changsoon (Voice Care Agent Project)

## 1. Basic Information

-   Persona Name: Changsoon
-   Age: 83
-   Living Situation: Lives alone in an urban apartment
-   Device Capability: Landline phone only

## 2. Context

-   Daily Routine: Watches TV during the day, limited outdoor activity
-   Environment: Quiet apartment, minimal social visits
-   Support Network: 2--5 adult children, daily check-in calls

## 3. Behavioral Traits

-   Communication Style: Slow-paced, prefers warm tone
-   Technology Usage Level: Very low
-   Cognitive Characteristics: Clear memory, but slow response speed

## 4. Emotional Context

-   Primary Needs: Companionship, safety reassurance
-   Fears / Concerns: Falling, medical emergency
-   Motivation: Wants emotional connection

## 5. Interaction Preference

-   Preferred Channel: Voice call
-   Conversation Length Preference: Moderate, especially at night
-   Tone Expectation: Caring, daughter-like warmth
