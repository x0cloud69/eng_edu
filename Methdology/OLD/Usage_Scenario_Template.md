# Example: Nighttime Emotional Check-in

## 1. Scenario Name

Night Loneliness Support

## 2. Situation Context

-   Time: 10:30 PM
-   Location: Changsoon's apartment
-   Emotional State: Feeling lonely

## 3. Trigger Event

No outgoing calls detected for 24 hours.

## 4. Agent Interaction Flow

1.  Agent initiates gentle greeting.
2.  Agent asks about her day.
3.  Agent suggests short breathing exercise.

## 5. Expected Outcome

-   Behavioral Change: Reduced loneliness indicators
-   System Response: Log emotional status

## 6. Human-in-the-loop Condition

-   If emergency keywords detected → escalate to family member
-   If no response after 3 attempts → alert monitoring system
