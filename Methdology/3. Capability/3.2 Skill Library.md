# 1. 개요

Skill은 AI Agent가 특정 작업을 수행하기 위해 호출하는 **캡슐화된 기능 단위**다. Tool이 외부 시스템(API, DB, 파일)에 접근하는 인터페이스라면, Skill은 Agent가 Tool을 활용해 목적을 달성하는 **실행 레시피**다.

> 💡 Skill = 목적 + Tool 조합 + 실행 순서 + 출력 형식. Agent는 직접 Tool을 호출하지 않고 반드시 Skill을 경유하여 Tool을 사용한다.

## 1.1 Skill 분류 체계

```
Skill 분류
├── Core Skills       — 모든 Agent 공통 사용 (로깅, 감사, 에스컬레이션)
├── Domain Skills     — 특정 Agent 전용 (코드 생성, UI 생성, 테스트 실행)
└── Composite Skills  — 2개 이상의 Skill을 조합한 복합 Skill
```

## 1.2 Skill 인터페이스 표준

모든 Skill은 아래 표준 인터페이스를 준수한다.

```python
class SkillBase:
    """모든 Skill의 부모 클래스"""
    skill_id: str          # 고유 식별자 (예: "core.audit_log")
    skill_name: str        # 사람이 읽을 수 있는 이름
    version: str           # 버전 (예: "1.0.0")
    allowed_agents: list   # 사용 허용 Agent 목록
    requires_approval: bool  # Human 승인 필요 여부
    audit_required: bool   # 감사 로그 기록 필수 여부

    async def execute(self, input: SkillInput) -> SkillOutput:
        """실행 메서드 — 모든 Skill이 구현"""
        raise NotImplementedError

    async def validate(self, input: SkillInput) -> bool:
        """실행 전 입력 유효성 검증"""
        raise NotImplementedError

    async def rollback(self, trace_id: str) -> bool:
        """실패 시 롤백 (해당되는 경우)"""
        return False
```

```python
    before_state: dict | None
    after_state: dict | None
    result: Literal["success", "failure", "pending"]
    metadata: dict = {}
```

**출력 스키마:**
```python
class AuditLogOutput(BaseModel):
    audit_id: UUID
    recorded_at: datetime
    success: bool
```

**사용 예시:**
```python
await skill_registry.execute("core.audit_log", AuditLogInput(
    trace_id=trace_id,
    agent_id="Back-A",

    target_type="migration",
    target_id=migration_id,
```

# 7. Skill 등록 및 호출 규칙

## 7.1 Skill Registry

```python
# skill_registry.py — 전체 Skill 등록 관리
class SkillRegistry:
    _skills: dict[str, SkillBase] = {}

    @classmethod
    def register(cls, skill: SkillBase):
        cls._skills[skill.skill_id] = skill

    @classmethod
    async def execute(cls, skill_id: str, input: SkillInput) -> SkillOutput:
        skill = cls._skills.get(skill_id)
        if not skill:
            raise SkillNotFoundError(skill_id)

        # 1. Agent 권한 확인
        if input.agent_id not in skill.allowed_agents:
            raise SkillPermissionError(skill_id, input.agent_id)

        # 2. 입력 유효성 검증
        await skill.validate(input)

        # 3. 승인 필요 여부 확인
        if skill.requires_approval and not input.approval_id:
            raise ApprovalRequiredError(skill_id)

        # 4. 실행
        output = await skill.execute(input)

        # 5. 감사 로그 (audit_required인 경우)
        if skill.audit_required:
            await cls.execute("core.audit_log", ...)

        return output
```
