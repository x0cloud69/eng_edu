# ✅ 예시 --- 다솜이 AI 가족 케어 플랫폼

## 📄 문서 정보

-   프로젝트명: 다솜이 AI 가족 케어 플랫폼
-   담당자: PM Agent
-   버전: v0.1

## 🎯 배경

한국은 빠르게 고령화되고 있으며 많은 노인이 독거 생활을 하고 있다.\
가족이 가까이 있어도 정서적 고립감은 여전히 높다.

## ❗ 핵심 문제

창순(83세)과 같은 고령 사용자는 스마트폰을 사용하지 못하고 먼저 연락을
시작하기 어려워\
특히 밤 시간대에 외로움과 안전 불안을 느낀다.

기존 서비스는 일상적 동행보다 긴급 대응 중심이다.

## 📊 근거 / 신호

-   TV 중심의 일상 루틴
-   낮은 발신 연락 빈도
-   저녁 시간대 정서적 취약성

## 🧭 문제 범위

**Included** - 음성 기반 동행 - 안전 확인 콜 - 가족 연결 브릿지

**Excluded** - 의료 진단 - 완전한 보호자 대체

**Non-Goals** - 복잡한 스마트폰 UX
